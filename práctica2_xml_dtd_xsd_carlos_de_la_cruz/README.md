# Trabajo de Validación XML


Este repositorio contiene la entrega dividida en tres partes, cada una con sus archivos XML, DTD y/o XSD según lo requerido.


## Estructura de carpetas


-  parte1/
- `catalogo.xml`
- `catalogo.dtd`
- `catalogo.xsd`


-  parte2/
- `plataforma.xml`
- `plataforma.dtd`
- `plataforma.xsd`


-  parte3/
- `library.xml`
- `library.dtd`
- `library.xsd`


Todos los archivos están escritos en formato texto plano (.xml, .dtd, .xsd)
